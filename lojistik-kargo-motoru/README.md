# Akıllı Lojistik ve Paket Fiyatlandırma Motoru

Bu proje, Python ile geliştirilmiş basit bir kargo ve lojistik fiyatlandırma sistemidir.

Kullanıcıdan ürün adı, kilo, boyut, en, yükseklik, mesafe ve gönderim türü bilgileri alınır.  
Program bu bilgilere göre gönderim türünü kontrol eder ve fiyatlandırma yapar.

## Kullanılan Yapılar

- input()
- float()
- if / elif / else
- dictionary
- boolean kontrolü
- temel hesaplama işlemleri

## Gönderim Türleri

- Standart
- Ekspres
- VIP
