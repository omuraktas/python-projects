# Python Projects

Bu repo, Python öğrenirken yaptığım projeleri ve alıştırmaları içerir.

## Projelerim

### 1. Akıllı Lojistik ve Paket Fiyatlandırma Motoru

Kullanıcıdan ürün adı, kilo, boyut, mesafe ve gönderim türü bilgilerini alarak kargo fiyatlandırması yapan Python projesi.

Klasör: `lojistik-kargo-motoru`

---

### 2. Öğrenci Bilgi Kartı Projesi

Kullanıcıdan öğrenci bilgileri, dersler, notlar ve etkinlik bilgileri alarak öğrenci bilgi kartı oluşturan Python projesi.

Klasör: `ogrenci-bilgi-karti`

---

### 3. BudgetMaster CLI

Gelir ve harcamaları takip etmek için geliştirilen menü tabanlı Python bütçe yönetim projesi.

Bu projede kullanıcı gelir ekleyebilir, harcama ekleyebilir, kategori analizi yapabilir, aylık bütçe özetini görebilir, tasarruf önerisi alabilir ve finansal skorunu hesaplayabilir.

Klasör: `budgetmaster-cli`
