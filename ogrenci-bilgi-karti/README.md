# Öğrenci Bilgi Kartı Projesi

Bu proje, Python ile geliştirilmiş basit bir öğrenci bilgi kartı sistemidir.

Program kullanıcıdan öğrencinin adı, soyadı, yaşı, boyu, mezuniyet durumu, staj durumu, vize, final ve proje notu gibi bilgileri alır.
Daha sonra bu bilgileri düzenleyerek öğrencinin bilgilerini ekrana yazdırır.

## Projenin Amacı

Bu projede amaç, Python'da kullanıcıdan veri alma, string düzenleme, liste kullanımı ve temel not hesaplama işlemlerini öğrenmektir.

## Kullanılan Yapılar

* input()
* int()
* list
* sort()
* lower()
* capitalize()
* print()
* temel matematik işlemleri
* değişken kullanımı

## Proje İçeriği

Programda kullanıcıdan şu bilgiler alınır:

* Öğrenci adı
* Öğrenci soyadı
* Yaş
* Boy
* Mezuniyet durumu
* Staj durumu
* Vize notu
* Final notu
* Proje notu
* Alınan dersler
* Etkinlik bilgileri

## Öğrendiklerim

Bu projeyi yaparken Python'da veri alma, string düzenleme, liste sıralama ve not hesaplama mantığını daha iyi öğrendim.
