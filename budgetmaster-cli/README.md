# BudgetMaster CLI

BudgetMaster CLI, Python ile geliştirilmiş basit bir bütçe ve harcama yönetim sistemidir.

Bu proje, kullanıcının gelirlerini ve harcamalarını kaydetmesini, harcamalarını kategorilere göre analiz etmesini, aylık bütçe özetini görmesini ve finansal durumunu değerlendirmesini sağlar.

## Projenin Amacı

Bu projede amaç, Python'da fonksiyonlar, listeler, sözlükler, tuple, set, koşul yapıları ve döngüler kullanarak menü tabanlı bir komut satırı uygulaması geliştirmektir.

## Özellikler

* Gelir ekleme
* Harcama ekleme
* Harcamaları listeleme
* Kategori analizi yapma
* Aylık bütçe özeti görüntüleme
* Acil durum simülasyonu yapma
* Tasarruf önerisi alma
* Finansal skor hesaplama
* Verileri sıfırlama

## Kullanılan Python Yapıları

* list
* dictionary
* tuple
* set
* function
* while loop
* for loop
* if / elif / else
* try / except
* random modülü

## Menü Seçenekleri

Program çalıştırıldığında kullanıcıya şu menü gösterilir:

```text
1. Gelir Ekle
2. Harcama Ekle
3. Harcamaları Listele
4. Kategori Analizi
5. Aylık Bütçe Özeti
6. Acil Durum Simülasyonu
7. Tasarruf Önerisi Al
8. Finansal Skorumu Gör
9. Verileri Sıfırla
0. Çıkış
```

## Öğrendiklerim

Bu projeyi yaparken Python'da fonksiyon kullanımı, veri saklama, kullanıcıdan veri alma, hata kontrolü, liste ve sözlük yapılarıyla çalışma konularını daha iyi öğrendim.

Ayrıca menü tabanlı bir programın nasıl kurulduğunu ve farklı fonksiyonların ana program akışında nasıl birlikte çalıştığını pratik ettim.
